# Deep Neural Networks

This folder contains the projects and assignments from the Deep Neural Networks course at the University of Warsaw. The course aims to provide practical knowledge of deep neural networks, covering currently used techniques, algorithms, and tools. The methods discussed are applied to real-world problems, particularly in the fields of image recognition (computer vision) and natural language processing.

## 📝 Syllabus

For detailed information on the course content, please refer to the official syllabus (in Polish):  
[Deep Neural Networks - Syllabus](https://usosweb.mimuw.edu.pl/kontroler.php?_action=katalog2/przedmioty/pokazPrzedmiot&prz_kod=1000-317bDNN)

## Task Organization
The tasks were given in the form of **Jupyter Notebooks**. Some tasks had empty spaces to be filled up, while others required us to create a new **Notebook**. Due to these discrepancies, for each task the **README.md** in the task root is exactly the **Notebook** given as an assingment, as a **Markdown** file. The results are in the **Project** subdirectories, in the form of **Jupyter Notebooks** and reports if necessary. You can run the code through **Google Colaboratory**; links are provided for each assingment.
