# Programming in Rust

This folder contains the project from the Programming in Rust course at the University of Warsaw. Rust is a modern, compiled programming language with a wide range of applications. The course explores its high performance (comparable to C or C++), memory safety without a garbage collector, strict and rich type system, functional programming idioms, and convenient concurrent programming.

## Organization

As part of the Programming in Rust course at University of Warsaw, at a request from our tutor - [Wojciech Przytuła](https://github.com/wprzytula), a ScyllaDB employee, me and 2 of my friends developed a tool for benchmarking, comparing, and visualizing the performance of different implementations, which we called **SDB** This folder only showcases what we were able to accomplish in the span of a few months and does not have any development history. Currently, the project is developed and maintained by ScyllaDB employees.
